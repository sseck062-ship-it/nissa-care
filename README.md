# Nisâa Care 🌙

**Application halal et privée pour le suivi du cycle féminin musulman**

### 🎯 Objectif
Aider nos sœurs à concilier 'ibada et santé intime en toute pudeur.

### ✨ Fonctionnalités V1
- Suivi simple du cycle menstruel
- Rappels de fiqh : ce qui est permis/interdit pendant les règles
- Guide du Ghusl 
- 100% privé : aucune donnée ne quitte le téléphone de la sœur

### 🤍 Pour Allah
Projet open-source pour la Oummah. Qu'Allah accepte.
